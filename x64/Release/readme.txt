exe запускается так: programm.exe first_image second_image save_path left_threshold rigth_threshold

programm.exe - сама программа

first_image - абсолютный путь первого изображения
Пример: C:/Users/Emil/Desktop/first.jpg

second_image - абсолютный путь второго изображения
Пример: C:/Users/Emil/Desktop/second.jpg

save_path - абсолютный путь для сохранения результатов

left_threshold - целочисленное значение левого порога

rigth_threshold - целочисленное значение правого порога
